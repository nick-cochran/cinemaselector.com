Cinema Selector

How many films to choose from?

The program randomly selects from a collection of films sorted by popularity. Select the number of films that will be randomly chosen from. If no number is selected, the most popular film will be chosen. Click on the dot before each selection to start recording (filter by that category). For more information about something, click on the info button.

Here you can select one genre to sort by.

Here you can set the range of years that will be selected from. If you just want to select from one year, set both handles to the same year.

Here you can set the range for how long/short the film is.

Type in an actor's name to the best of your abilities and the program will try to find the actor you are looking for and then show you only films that they are in.

Type in a crew member's name to the best of your abilities and the program will try to find the crew member you are looking for and then show you only films that they worked on. This includes directors, writers, producers, etc.

Here you can select one streaming service to sort by.
